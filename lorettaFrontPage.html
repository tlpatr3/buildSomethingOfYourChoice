Murder in the Mountains: Who Killed Loretta Lynn Willoughby?

This website is a collection of newspaper articles from the Irvine, Kentucky, area regarding the death of Loretta Lynn Willoughby.

Please note that this website is a work in progress. It is intended to inform the public about the case of Loretta Willoughby in Irvine, Kentucky. If you have any information about this case, please contact the Kentucky State Police Post 7 in Richmond, Kentucky, at 859-623-2404.
